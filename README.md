Online Food Ordering System Project in PHP


Type of users/Actors

  1.	Admin --> admin 
  2.	Vendor --> cheezios
  3.	Customer --> Faraz



  
Admin Features

  1.	Login
  2.	Manage Vendors
  3.	Manage Vendors Products/Food
  4.	Manage Customers
  5.	Manage Food Order details
  6.	Manage Payment details
  
Vendor Features

  1.	Create account
  2.	Login
  3.	Add Food details
  4.	Update Food details
  5.	Delete Food
  6.	Check food order
  7.	Update food delivery status
  
Customer Features

  1.	View Food details
  2.	Register/Create Account
  3.	Login
  4.	Add to cart
  5.	Update Cart
  6.	Checkout
  7.	Pay Cash On Delivery
  8.	Check Food delivery status
